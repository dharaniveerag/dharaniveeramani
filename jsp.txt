1.what is jsp?
*JSP technology is used to create web application just like Servlet technology. 
*It can be thought of as an extension to Servlet because it provides more functionality than servlet such as expression language, JSTL.
*A JSP page consists of HTML tags and JSP tags. 
*The JSP pages are easier to maintain than Servlet because we can separate designing and development.
*It provides some additional features such as Expression Language, Custom Tags.
